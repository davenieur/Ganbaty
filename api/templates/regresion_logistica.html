    2. Regresión logística

    ¿Qué es la regresión logística?

    Es otro tipo de algoritmo de aprendizaje supervisado cuyo objetivo es predecir valores binarios (0 o 1). Este algoritmo consiste en una transformación a la regresión lineal. La transformación se debe a que una regresión lineal no funciona para predecir una variable binaria, para esta transformación se emplea la siguiente función sigmoide:

    \( \widehat{y} = \frac{1}{1+e^{-(a+bX)}} \)

    Donde: e = Número de Euler (2.718).
